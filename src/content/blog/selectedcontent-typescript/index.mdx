Yesterday I was coding an app with solid and I wanted to use the new `selectedcontent` html element that is available inside `<select>` elements. Combined with some CSS it enables you to be able to style dropdowns in any way you want. `selectedcontent` specifically allows you to have custom html within the options and be able to style the selected option:

<SelectExample isModern={true} />

<Code
    lang="jsx"
    title="LanguageSelector.tsx"
    code={`
export const LanguageSelector = () => {
    return (
        <select id="languageSelection">
            <button>
                <selectedcontent />
            </button>
            <option value="en">
                <span class="icon" aria-hidden="true">
                    🇬🇧
                </span>
                English
            </option>
            {...other options}
        </select>
    );
};`}
/>

```css title="index.css"
::picker(select),
select {
    appearance: base-select;
}

selectedcontent {
    display: flex;
    flex-direction: row-reverse;
    gap: 1rem;
}
```
